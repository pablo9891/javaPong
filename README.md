# Java Pong

Java Pong is a first game attempt. All written using Java Swing

![JavaPong](images/pong.gif)

## Build

```bash
mvn clean package
```

## Usage

From your target folder
```bash
java -jar JavaPong-1.0-jar-with-dependencies.jar
```
